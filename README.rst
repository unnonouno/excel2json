============
 excel2json
============

A simple script to convert an Excel CSV file to a json.


Usage
=====
::

   excel2json [-h] [-d] FILE

Convert Excel CSV file to a JSON.

positional arguments:
  FILE        excel CSV file to open.

optional arguments:
  -h, --help  show this help message and exit
  -d, --dict  print as a dictionary.
